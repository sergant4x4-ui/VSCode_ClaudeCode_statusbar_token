# Claude Token Status

Небольшое расширение для VS Code, которое показывает в статус-баре (внизу окна)
заполненность контекстного окна текущей сессии Claude Code — сколько токенов
"весит" последний ответ относительно лимита в 200 000 токенов.

Пример: `$(pulse) 48.2k/200k tok (24%)`

Наведите курсор на индикатор, чтобы увидеть разбивку: input / output / cache write / cache read.

## Что это НЕ показывает

Это не 5-часовой лимит использования Claude (тот виден только через команду `/usage`
в самом Claude Code). Индикатор показывает только заполнение контекстного окна
текущего диалога — величину, после которой Claude Code сам сожмёт историю (compact).

## Установка

1. Скачайте файл `claude-token-statusbar.vsix` из этого архива.
2. Откройте VS Code.
3. Откройте палитру команд: `Ctrl+Shift+P`.
4. Введите **Extensions: Install from VSIX...** и выберите скачанный файл.
5. Перезагрузите окно (`Ctrl+Shift+P` → **Developer: Reload Window**).
6. Внизу окна (статус-бар) появится индикатор — расширение работает, если на
   компьютере установлен и хотя бы раз запускался Claude Code (нужны файлы
   сессий в `~/.claude/projects/`).

Либо через терминал:

```
code --install-extension claude-token-statusbar.vsix
```

## Требования

- VS Code 1.85 или новее.
- Установленный Claude Code (CLI или расширение) — индикатор читает его файлы
  сессий и без них скрывает себя.
